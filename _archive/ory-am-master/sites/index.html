HOW IT WORKS

Installation

Install ORY Sites with one command. NodeJS needs to be installed on your system. Please be aware that ORY Sites for npm is not yet available for the public.

$ npm install -g ory-sites

Getting started

Create and open a project with ease using the command line interface. The project will open in your browser. Hot reloading, debugging tools and other useful utilities are enabled by default.

$ ory-sites create my-project
$ cd my-project
$ ory-sites start
Please open your browser at http://localhost:4928

Build and deploy static site

Building and publishing your site is a matter of seconds. The result is static HTML, JavaScript and CSS.

We support different hosting providers and will keep on adding new ones.

$ ory-sites build
Build successful!

$ ory-sites deploy netlify .
Site now live at at example.com!

Sane output

It is important that HTML source code is sane and readable. It will improve SEO and browser compatiblity.

Forget horrific WYSIWYG output, we use a sophisticated JSON document to save and transform content.
